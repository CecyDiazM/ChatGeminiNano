✍️ Escritor con IA (AI Writer)
Escritor con IA is a sleek, mobile-optimized writing assistant designed to generate structured, context-aware content on demand. Boasting a clean and minimalist dark user interface, the application enables users to input a writing prompt, customize the tone and length, and quickly copy the generated text with a single tap.

🚀 Key Features
Customizable Parameters: Tailor your output with easy-to-use dropdown menus for text tone (e.g., Formal, Neutral) and text length (e.g., Short, Medium, Long).

Smart Backoff & Retry Handling: Includes built-in error handling that automatically retries generation and displays a countdown timer if servers are busy ("Servidores ocupados, reintentando en 2s...").

Markdown Formatting Support: Outputs beautifully structured, human-like text complete with bold headers, bullet points, and clean lists—ideal for professional CV templates, emails, or blog drafts.

One-Click Clipboard Integration: Features a dedicated "Copiar texto" button at the bottom of the output that seamlessly changes state to "¡Copiado!" upon success, making exporting your content effortless.

Responsive Mobile-First Design: Crafted with modern web styling constraints to ensure a premium look and comfortable experience on mobile browsers and desktop devices alike.

🧠 Problems It Solves
Writer's Block & Blank Page Syndrome: Quickly generates comprehensive initial drafts (such as professional Full-Stack Web Developer CV outlines) based on simple, raw tech stack inputs.

Inconsistent Content Tone: Adjusting a text manually to fit different contexts can be time-consuming. This app helps users easily pivot the voice of their text from casual to highly formal in seconds.

Cluttered Interfaces and Complicated Workflows: Avoids the overwhelming sidebars, prompt engineering requirements, and login popups found in other platforms by focusing on a direct, streamlined text generation box.

Unstable API Errors: Many AI apps break completely when an API limit or server overload occurs. The built-in retry logic ensures a smoother user experience during high-traffic windows.
